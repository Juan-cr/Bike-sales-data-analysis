# Bike Sales Data Analysis (Excel Project)

## Project Title

**Bike Sales Data Analysis**
This project analyzes a bike sales dataset using Microsoft Excel to identify customer purchasing behavior, sales trends, and underperforming markets. The goal is to simulate real-world business decision-making by transforming raw data into actionable insights.

## Business Understanding

The objective of this project was to analyze bike sales data to uncover trends and identify opportunities for improving business performance.

The main goal was to determine:

* Which customer segments generate the most revenue
* Which regions are underperforming
* What products perform best

This project was selected as part of a data analytics training program to practice real-world analytical thinking using Excel.

One challenge encountered was transforming raw product data into structured categories (such as model, size, and color), which required additional data cleaning and transformation.

---

## Data Understanding

The dataset contains approximately **90 records** and includes the following variables:

* Sales Order Number
* Purchase Date
* Customer Age & Age Group
* Customer Gender
* Country & State
* Product Category, Model, Color, Size
* Order Quantity
* Unit Cost, Unit Price
* Revenue, Cost, Profit

This dataset was chosen because it simulates a real retail business scenario, allowing analysis of both customer demographics and product performance.

## Approach

The project followed a structured data analytics process:

1. **Data Cleaning**

   * Removed duplicate records
   * Handled missing values
   * Standardized formats (dates and text)
   * Split product attributes using Text to Columns

2. **Data Transformation**

   * Organized product details into structured fields
   * Prepared data for analysis using Excel functions

3. **Data Analysis**

   * Used Pivot Tables to analyze sales by country, gender, and age group
   * Identified high-performing and low-performing segments

4. **Data Visualization**

   * Built an interactive dashboard with KPIs and charts
   * Included bar, column, and pie charts for clear insights

5. **Insight Generation**

   * Identified key revenue drivers and customer behavior patterns
   * Provided business recommendations based on findings
